# Task Manager App

A full-stack web application built using Angular and .NET Web API.

## Features
- Add and delete tasks
- REST API integration
- Real-time UI updates

## Tech Stack
- Frontend: Angular (Standalone)
- Backend: .NET Web API
- Database: SQLite

## Key Learnings
- API integration using HttpClient
- Handling CORS issues
- Angular standalone architecture
- Debugging dependency injection issues
